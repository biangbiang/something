homebrew学习笔记
===============

### 国内镜像

Mac 终端跑入即可

    cd /usr/local/  
    git remote set-url origin http://mirrors.ustc.edu.cn/homebrew.git  
    或者这个：
    git remote set-url origin git://mirrors.tuna.tsinghua.edu.cn/homebrew.git  
    brew update  

如果还是感觉慢，看看这个：

    http://mirrors.tuna.tsinghua.edu.cn/help/#homebrew
